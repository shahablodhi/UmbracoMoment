Release notes for Kendo UI Core v2017.3.1026

Components

What's fixed

  • detailinit is always evaluated in vue wrapper (#3676)
  • Overlay distorts over disabled widgets in Chrome (#3661)
  • Escape calc() functions that use variables in less themes to prevent wrong compiling (#3638)
  • Kendo typings error using TypeScript 2.5.2 -> DataSourceTransportWithFunctionOperations (#3550)

AutoComplete

What's fixed

  • Incorrect AutoComplete popup position (#3602)
  • fixedGroupTemplate not displayed in AutoComplete, ComboBox, DropDownList and MultiSelect with virtualization enabled (#3548)
  • Widget's dropdown auto-closes in Chrome when a preceding element's height is reduced (#3516)
  • DropDownList / ComboBox/ AutoComplete items filter is not cleared, when certain item was selected (base on search criteria) and minLength options is set (#2950)
  • AutoComplete, MultiSelect and ComboBox clearButton option can not be set via setOptions() (#2222)

  • Widget's dropdown auto-closes in Chrome when a preceding element's height is reduced (#3516)
  • When using Grid custom editor, the Kendo Editable adds data-binding for the value for both the hidden and the visible inputs of the ComboBox. (#3427)
  • ComboBox setOptions applies dataTextField and dataValueField, but the widget shows the new fields as undefined (#3218)
  • DropDownList / ComboBox/ AutoComplete items filter is not cleared, when certain item was selected (base on search criteria) and minLength options is set (#2950)
  • AutoComplete, MultiSelect and ComboBox clearButton option can not be set via setOptions() (#2222)
